# FOPCPD Database Example
This is a sample file structure to showcase what the final system may resemble. As this is a draft, all values are fake and void of depth.

To view either the Community.xlsx spreadsheet example or the .docx Word version of the Email Example, you must download the raw file (GitHub doesn't support displaying either file type). To do so, simply open the file on this repository, and click 'view raw' to download it.

## Navigation
| File Name | Description |
| --- | --- |
| .gitignore | Core Git/GitHub file required for proper initialization of this repo; please disregard it. |
| Community.xlsx | Sample spreadsheet of a couple of formatted tables representing distinct groups/organizations. More info when viewing the file. |
| EMPTY_Foundation.xlsx | Blank of any information; simply there to highlight the ability to have multiple spreadsheets. Please disregard it. |
| EMPTY_Political.xlsx | Blank of any information; simply there to highlight the ability to have multiple spreadsheets. Please disregard it. |
| Email Example.docx | Same as below, but in Word format if preferred (requires manual download) |
| Email Example.md | Sample email that showcases what would be uploaded to Power Automate with variable examples included (the items in '<>'). These variables would systematically be replaced by each recipient's personal data values from the spreadsheets. |
| README.md | The file you are currently viewing! |
